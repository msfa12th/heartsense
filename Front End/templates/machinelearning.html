Machine Learning Pipeline for Predictive Analysis of Heart Disease

The machine learning model is built using Support Vector Classifier algorithm with hyperparameters (‘C’: 5, ‘gamma’: 0.005). The accuracy of the model is 73%. Initial model was trained on a random subset of data and was tested against a subset of test data. This model is for predictive purposes only and should not be used as medical advice.
